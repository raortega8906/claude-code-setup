# Lecciones aprendidas del proyecto

> Este archivo es CLAVE. Cada vez que Claude comete un error y lo corriges,
> añade aquí una regla concreta para que no vuelva a pasar.
> Claude lee este historial y adapta su comportamiento.

---

## Cómo añadir una lección

Usa el comando `/lesson [descripción del error]` o pide directamente:
```
Añade a tasks/lessons.md la regla para evitar [error que acabamos de corregir].
```

---

## Formato de cada entrada

```
## YYYY-MM-DD — Título breve del problema
**Problema:** Qué salió mal
**Causa:** Por qué ocurrió
**Solución:** Cómo se arregló
**Regla:** INSTRUCCIÓN CONCRETA para sesiones futuras
```
